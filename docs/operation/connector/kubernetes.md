# Kubernetes

## 创建 Kubernetes 连接器

1. 进入连接器列表页。
2. 点击`新建连接器`按钮，选择 Kubernetes。
3. 输入集群名称和KubeConfig。
4. 选择是否开启`启用成本分析`。开启后可在左侧菜单`成本管理`中可创建相关视图查看统计相关的消费金额；
5. 点击`保存`即可创建 Kubernetes 连接器。

## 查看、编辑连接器

1. 进入连接器列表页。
2. 找到您要查看或编辑的连接器。
3. 点击列表中的连接器名称可查看详情；
4. 点击`编辑`操作对连接器进行编辑。

## 删除连接器

> 注：在删除连接器前，需要确认当前没有环境依赖该连接器。

1. 进入连接器列表页。
2. 选中您要删除的连接器。
3. 点击`删除`按钮，确认即可删除。
