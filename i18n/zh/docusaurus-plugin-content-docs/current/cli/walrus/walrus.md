# walrus

Command line interface for Walrus

## Examples

```

  # Setup Walrus CLI configuration
  $ walrus config setup

```

## Options

```
  -d, --debug           Enable debug log
  -h, --help            Help for this command
  -o, --output string   Output format [table, json, yaml] (default "table")
  -v, --version         Version for CLI
```
